1. Введение

IridiumPY — интерпретируемый язык программирования с простым синтаксисом, вдохновлённым Python. Он позволяет писать скрипты с минимальными конструкциями, но использовать мощь Python через встроенные команды.

  • Расширение файлов: .ird
  • Основные возможности: переменные, условные конструкции, ввод/вывод, Python-библиотеки
  • Комментарии: // для однострочных, /* ... */ для многострочных

2. Синтаксис языка

2.1 Переменные и типы

a = 10           // число
b = 3.14         // float
name = "Alice"   // строка
list = [1,2,3]   // список
dict = {a: 1, b: 2} // словарь

2.2 Ввод и вывод

println("Hello")           // вывод с переносом строки
print("Hello")             // вывод без переноса
age = coninp("Your age: ") // ввод пользователя

2.3 Условные конструкции

if age < 14 {
    println("Too young!")
}
elif age > 14 {
    println("Good, your age is ")
    print(in"{age}")
}
else {
    println("Exactly 14!")
}

2.4 Работа с Python-библиотеками

pyuse "math"           // импорт библиотеки
pycode>> a = math.cos(1) // выполнение Python-кода
print(in"{a}")      // полчуение переменной из Python-кода

3. Примеры

3.1 Hello World

println("Hello, world!")

3.2 Ввод возраста

age = int(coninp("Your age: "))
if age >= 18 {
    println("Adult")
}
else {
    println("Minor")
}

3.3 Использование Python-библиотеки

pyuse "time"

println("Wait 2 seconds...")
pycode>> time.sleep(2)
println("Done!")

4. Сборка .exe (Windows)

4.1 Трансляция и сборка

iridium_translate file.ird file.py
pyinstaller --onefile --console file.py
  • --onefile — один .exe файл
  • --console — вывод программы в терминал

4.2 Запуск .exe

.\dist\file.exe

Через двойной клик консоль откроется и закроется после выполнения. Чтобы окно не закрывалось:

pycode>> input("Press Enter to exit...")

5. Расширенные возможности языка

  • Работа со списками и словарями
  • Циклы for и while
  • Встроенные Python функции через pycode>>
  • F-строки для форматированного вывода
  • Многострочные выражения через блоки {} и автоматическая табуляция
